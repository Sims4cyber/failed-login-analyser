# Failed Login Analyser

## Overview

This project was completed as part of my cybersecurity learning journey.

The script analyses login activity and identifies failed login attempts from log data. It demonstrates fundamental Security Operations Centre (SOC) skills including log analysis, threat detection and Python automation.

## Skills Demonstrated

- Python
- Log Analysis
- Security Monitoring
- Cybersecurity Fundamentals
- Data Processing
- Incident Detection

## Project Objective

The goal of this project is to automate the identification of failed login attempts within a log file and provide a summary of suspicious authentication activity.

## How It Works

The script:

1. Reads login activity data from a log file
2. Identifies failed login attempts
3. Counts failed login attempts per user
4. Generates a summary report
